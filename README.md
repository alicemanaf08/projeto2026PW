### Tema
Sistema de hotel / pousada.

---

### Objetivo
Um sistema desenvolvido para ajudar no cadastro e reservas de clientes no hotel ou pousada.

---

### Ferramentas
* A definir

---

### Funcionamento
O sistema contara com o seguinte funcionamento:

1. **Cadastro simples dos clientes:**
   * Nome
   * Telefone
   * E-mail
   * CPF

2. **Informações sobre os quartos:**
   * Status (reservados e livres)
   * Quantidade de camas
   * Andar localizado
   * Numero do quarto

3. **Controle de Reservas:**
   * Atribuira a reserva a um codigo unico que corresponde ao cliente cadastrado.
